# 🛍️ Ecommerce Shop - Nike Store

Welcome to the **Ecommerce Shop** project! This is a modern, responsive ecommerce website designed specifically for showcasing and selling Nike shoes. Built with pure HTML, CSS, and JavaScript, it provides a sleek user experience for browsing products, viewing features, and making purchases.

## ✨ Features

- 🏃‍♂️ **Product Slider**: Interactive slider showcasing various Nike shoe models like Air Force, Air Jordan, Blazer, Crater, and Hippie.
- 📦 **Shipping & Returns**: Highlighted features for free shipping, 30-day returns, and gift cards.
- 💳 **Secure Payments**: Integrated payment form with Visa and Mastercard support.
- 📱 **Responsive Design**: Optimized for all devices, from mobile to desktop.
- 🎨 **Modern UI**: Clean, stylish design with smooth animations and transitions.
- 🌐 **External Images**: All images sourced from free online repositories for fast loading.
- 📧 **Newsletter Signup**: Footer section for subscribing to updates.
- 🔗 **Social Links**: Direct links to social media platforms.

## 🚀 How to Run

1. 📂 Clone or download the project files.
2. 🌐 Open `index.html` in your favorite web browser.
3. 🛒 Start browsing the Nike shoe collection!

No additional setup or dependencies required – it's a static website!

## 📁 Project Structure

- `index.html` - Main HTML file with the website structure.
- `style.css` - CSS styles for layout, colors, and responsiveness.
- `script.js` - JavaScript for interactive features like the slider and payment form.

## 🛠️ Technologies Used

- **HTML5** 📄 - For semantic markup.
- **CSS3** 🎨 - For styling and animations.
- **JavaScript** ⚡ - For dynamic functionality.
